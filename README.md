# 💰 Sistema Financeiro em PHP

Sistema web simples para controle financeiro desenvolvido em PHP procedural com MySQL.

O projeto conta com autenticação de usuários, dashboard e CRUD completo de transações (entradas e saídas).

---

## 🚀 Tecnologias

- PHP (procedural)
- MySQL
- HTML / CSS
- Bootstrap
- Sessions (autenticação)

---

## 📦 Funcionalidades

### 🔐 Autenticação
- Login com email e senha
- Sessão com `$_SESSION`
- Proteção de rotas
- Logout

---

### 📊 Dashboard
- Página inicial após login
- Acesso rápido às transações
- Navegação simples

---

### 💰 Controle Financeiro

CRUD completo de transações:

- ➕ Criar transação
- 📄 Listar transações
- ✏️ Editar transação
- ❌ Excluir transação

Tipos de transação:
- Entrada
- Saída
